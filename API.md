## `canvasLineChart`

Draw a line chart on canvas.

### Parameters

* `c` **`Canvas`** canvas element
* `height` **`number`** 
* `width` **`number`** 
* `data` **`Array<Array<number>>`** , as [zoom, val] doubles
* `base` **`number`** mathematical base, a number between 0 and 1
* `options` **`Object`** optional customizations
